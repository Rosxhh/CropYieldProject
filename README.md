# CropYieldProject
# 🌾 Crop Yield & Soil Analysis AI System

## 📌 Project Overview

The **Crop Yield & Soil Analysis AI System** is an AI-powered web application designed to help farmers and researchers analyze soil conditions and predict crop yield.
The system uses **Machine Learning and Deep Learning models** to classify soil types and provide crop recommendations.

This project aims to improve agricultural productivity by combining **Artificial Intelligence with modern web technologies**.

---

## 🚀 Features

* 🌱 **Soil Type Classification**
  Classifies soil images into different soil categories using deep learning models.

* 📊 **Crop Yield Prediction**
  Predicts crop yield based on soil properties and environmental factors.

* 🌾 **Crop Recommendation System**
  Suggests suitable crops depending on soil conditions.

* 🤖 **AI Diagnostic Tools**
  Includes utilities for verifying and testing trained models.

* 💬 **Chatbot Support**
  Provides user assistance and agricultural guidance.

## 📊 Dataset

This project uses the **Comprehensive Soil Classification Dataset** available on Kaggle.

Dataset Source:
https://www.kaggle.com/datasets/ai4a-lab/comprehensive-soil-classification-datasets

The dataset contains multiple soil categories including:

* Alluvial Soil
* Black Soil
* Red Soil
* Yellow Soil
* Laterite Soil
* Mountain Soil
* Arid Soil

These images are used to train deep learning models for soil classification.

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/Rosxhh/CropYieldProject.git
```

Navigate to the project folder:

```bash
cd CropYieldProject
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

Run the development server:

```bash
python manage.py runserver
```

Open in browser:

```bash
http://127.0.0.1:8000
```
